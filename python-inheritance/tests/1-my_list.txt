The ``MyList`` class
====================

Import the class:
    >>> MyList = __import__('1-my_list').MyList

Initialize the class and add elements:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)

Check the original list:
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Check the sorted output:
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

Verify the original list is still unchanged:
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Test with an empty list:
    >>> my_list = MyList()
    >>> my_list.print_sorted()
    []

Test with negative numbers:
    >>> my_list = MyList([1, -5, 10, 0])
    >>> my_list.print_sorted()
    [-5, 0, 1, 10]
